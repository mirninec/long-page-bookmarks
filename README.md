# 📖 long-page‑bookmarks

**Вертикальные закладки для длинных веб‑страниц**

Удобный userscript, который добавляет компактную вертикальную панель закладок справа на любой странице.  
Идеальное решение для чтения длинных статей, документации, логов и бесконечных списков.

## ✨ Зачем это нужно?

Многие страницы в интернете очень длинные. Легко потерять место, где вы остановились.  
Этот скрипт помогает быстро перемещаться по странице и **запоминать позицию чтения** даже после перезагрузки.

## 🚀 Что умеет скрипт?

- Автоматически создаёт закладки в зависимости от высоты страницы  
- Показывает визуальный прогресс каждой закладки с помощью градиента  
- Подсвечивает активную закладку зелёным цветом  
- Позволяет сохранить текущую позицию как активную закладку  
- Поддерживает прямые ссылки с хэшем (`#bookmark3`)  
- Плавно прокручивает страницу при клике на закладку  
- Сохраняет выбранную закладку в браузере для каждой страницы отдельно  
- Панель аккуратно синхронизируется при прокрутке и изменении размера окна  

## 🛠️ Как установить

### Рекомендуемый способ — Tampermonkey / Violentmonkey

1. Установите расширение **[Tampermonkey](https://www.tampermonkey.net/)** (Chrome, Edge, Firefox) или **Violentmonkey**.  
2. Создайте новый скрипт в расширении.  
3. Скопируйте содержимое файла [`bookmark.js`](bookmark.js) и вставьте вместо всего текста.  
4. Сохраните скрипт.

Готово! Скрипт будет автоматически работать на всех сайтах.

### Альтернативный способ — через консоль браузера

1. Откройте нужную страницу.  
2. Нажмите **F12** → вкладка **Console**.  
3. Вставьте весь код скрипта и нажмите `Enter`.

> ⚠️ При этом способе закладки работают только до перезагрузки страницы.

## 📖 Как пользоваться

1. На странице появится **тонкая вертикальная панель** справа.  
2. Кликните по номеру закладки — страница плавно прокрутится к этой позиции.  
3. Наведите курсор на любую закладку и подождите около секунды — появится кнопка **«Установить закладку»**.  
4. Нажмите кнопку — текущая позиция страницы станет активной (подсветится зелёным).  
5. Активная закладка сохраняется автоматически и восстанавливается при следующем посещении страницы.

**Полезная фишка:**  
Можно делиться точной позицией с помощью ссылки вида:

```text
https://example.com/article.html#bookmark5
```

## 🎯 Особенности

- Работает на большинстве современных сайтов  
- Сохраняет закладки отдельно для каждой страницы  
- Не требует внешних библиотек  
- Лёгкий и ненавязчивый дизайн  
- Поддержка хэшей в URL для удобного обмена ссылками
